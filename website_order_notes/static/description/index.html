Website Order Notes

Improve customer communication in Odoo!

Odoo Website Order Notes allows customers to add a message or note to the order. This helps the customer in conveying their message regarding the placed order. Additionally, this module let you conveniently receive messages or notes in Odoo.

Why Odoo Website Order Notes are Important?

Odoo Website Order Notes are important for your website because it allows you and your customer to connect via one-to-one chat. Also, it let customers add notes in the order by which they can add delivery instructions, packing requirements, or anything else.

Moreover, order notes help you know what customer wants and allow you to improve your website. And it also aids you in understanding the customer's requirements and serving them properly according to them.

Add Messages or Notes in Odoo

  • Odoo Website Order Notes permit customers to add messages or notes to the placed order.
  • You can also add notes or messages to the sales order from the Odoo backend.

    • Read Customer's Messages in Order Quotation

  • You can read every Customer's message under quotation in the Odoo backend.
  • Also, under stock picking, you can see the notes.

    • Read Messages At Your Convenience
  • You can read the message or note at the time of invoicing.

    • Select Desired Delivery Date

  • Along with order notes or messages, customers can also choose the desired delivery date.
